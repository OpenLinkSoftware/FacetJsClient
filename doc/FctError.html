Class: FctError

FctError(message, httpStatusText, httpStatusCode, responseText, responseXmlopt)

Represents a Facet error reported by the Facet service.

Constructor

new FctError(message, httpStatusText, httpStatusCode, responseText, responseXmlopt)

responseText may contain XML. If so, this is contained as parsed XML in responseXml. responseXml is converted to a JS object contained in responseJson. responseJson represents a Facet server error, described by property 'error'.
  • error - An object containing the following properties:
    • code - The Virtuoso error code.
    • message - A short server message.
    • diagnostics - Detailed information about the error source.
Parameters:
Name Type Attributes Description
message string Error message.
httpStatusText string The HTTP status text corresponding to the HTTP status code.
httpStatusCode number The HTTP status code returned by the server.
responseText string The response body as text.
responseXml string <optional>
 The response body as XML (if available)
